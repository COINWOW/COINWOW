COINWOW Core (CW)

COINWOW Core is the reference implementation of the COINWOW cryptocurrency network.

Website: https://coinwow.ca

Overview

COINWOW (CW) is a decentralized cryptocurrency inspired by Bitcoin and built around simplicity, transparency, and long-term reliability.

COINWOW Core connects to the COINWOW peer-to-peer network, validates blocks and transactions, and provides a secure wallet and graphical user interface for users and node operators.

Features

* Independent blockchain
* SHA256d Proof-of-Work
* Full node support
* Peer-to-peer network
* DNS Seed support
* Windows Qt Wallet
* Open source software
* MIT License

COINWOW Core Components

COINWOW Core includes:

* Full Node (coinwowd)
* Command Line Interface (coinwow-cli)
* Graphical Wallet (coinwow-qt)
* Wallet Utility (coinwow-wallet)

Building

Build instructions for Linux and Windows are available throughout the documentation and source tree.

Linux

cmake -B build
cmake --build build -j$(nproc)

Windows

COINWOW Core supports Windows builds through the MinGW and Qt toolchain.
